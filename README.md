# DivyOS

DivyOS is a fictional operating system built around my projects, resume, GitHub, and journey as a developer.

It is not a standard portfolio website. The goal is to make visitors forget they are inside a browser.

## Philosophy

DivyOS must never become a clone of Windows, macOS, or GNOME. It has its own original operating system identity. The UI/UX is intentionally designed to be premium, avoiding generic dashboards, templates, and "AI slop".

Every application within DivyOS has its own unique visual identity and architecture.

## Development

DivyOS is built using React 19, TypeScript, Vite, TanStack Router, Tailwind CSS v4, Framer Motion, and Zustand.

### Running Locally

```bash
npm install
npm run dev
```

### Building for Production

```bash
npm run build
```
