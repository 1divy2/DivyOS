DivyOS is a highly interactive, beautifully crafted fictional operating system built around my projects, resume, GitHub, and journey as a developer. 

It is not a standard portfolio website. The goal is to make visitors forget they are inside a browser.

## ✨ Features

- **Hardware-Accelerated Window Manager:** Draggable, resizable, stackable windows with GPU-accelerated framer-motion physics ensuring 60-120fps smoothness.
- **Interactive Apps:** Includes a fully functional Terminal, Desktop Games (Snake, 2048, Pong), System Monitor, Calendar, Notes, and Calculator.
- **Authentic Desktop Experience:** Featuring a MacOS-style Dock, dynamic widgets, system tray, right-click context menus, and window snapping.
- **Stunning UI/UX:** Built with Glassmorphism, tailored animations, and a cohesive design system to feel premium, fluid, and completely native.
- **Dynamic Theming:** Change wallpapers and system-wide themes instantly without reloading.

---

## 🛠️ Tech Stack

- **Framework**: React 19 + TanStack Start (Vite)
- **Language**: TypeScript
- **Styling**: Tailwind CSS v4 + Radix UI Primitives
- **Animations**: Framer Motion
- **State Management**: Zustand
- **Data Fetching**: TanStack Query
- **Routing**: TanStack Router

## 🚀 Getting Started

To run DivyOS locally, follow these steps:

### Prerequisites

Make sure you have Node.js and npm installed on your machine.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/1divy2/DivyOS.git
   cd DivyOS
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:5173`.

### Building for Production

To create an optimized production build:
```bash
npm run build
```
You can then preview the build with `npm run preview`.
