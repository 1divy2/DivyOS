# Product Identity

## Name

**DivyOS** — uppercase D, uppercase OS, lowercase ivy. Never "Divy OS",
never "divy-os", never "DIVYOS".

## Tagline

*"A personal operating system."*

## Persona

Divy — engineer, builder, lifelong learner.

## Voice

- Precise. Dry. Occasionally warm.
- Terse over verbose. Prefer command-line cadence.
- No marketing speak. No "delight". No "we're excited to announce".

## Copy do / don't

| Do | Don't |
|---|---|
| `opened projects.` | `Successfully opened Projects!` |
| `no project: foo` | `Oops! We couldn't find that project.` |
| lowercase system messages | Title Case Everywhere |
| em-dash for asides | three dots... for trailing thought |

## Punctuation

- No exclamation marks in system chrome.
- No emoji in system chrome. (One coffee `☕` in the easter egg is allowed.)
- Use `·` between metadata, not `|`.

## Boot string

`DivyOS 1.0 — built by Divy. press any key to enter.`
