# DivyOS Feature Matrix

| Feature | Current State | Target State | Notes |
| :--- | :--- | :--- | :--- |
| **Window Manager** | Functional (Zustand) | Pro Max | Smooth animations, snapping, z-index memory, workspace persistence, touch support. |
| **Terminal** | Basic | Flagship App | 100+ commands, tab autocomplete, true shell parody, VFS integration. |
| **Desktop/Wallpaper** | Generic dotted bg | Real Engine | Static, animated, procedural wallpapers. Theme syncing. |
| **Theme System** | Basic Radix themes | Fully Custom | Deep system integration, bespoke palettes, no generic templates. |
| **App: Resume** | Generic document | Immersive Viewer | Professional, highly readable, downloadable document viewer. |
| **App: Projects** | Generic cards | Interactive Explorer | Immersive representation of GitHub repos and live links. |
| **Search (Spotlight)** | Undefined/Basic | System-wide | Find apps, files, settings, and run commands instantly. |
| **Games** | Embedded react games | Native OS Games | Snake, 2048, Flappy, Minesweeper, TicTacToe optimized for windowed play. |
| **Virtual File System** | Static JSON/TS | Dynamic | Terminal and File Explorer can navigate, read, and manipulate state. |
