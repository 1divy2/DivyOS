# whoami

Hi — I'm **Divy**. I build software.

This is **DivyOS**, a personal operating system I made to share my work
without putting it inside another generic portfolio template. Everything
you see is a real application. Try the **Terminal** (type `help`), or
hit **⌘K** to search.

> Most of this content is a placeholder while the OS itself is the
> deliverable. Projects below are real — pulled live from my GitHub
> ([@1divy2](https://github.com/1divy2)).

## what's inside

- A window manager with drag, resize, snap, focus, persistence.
- A terminal with 100+ commands. The terminal *is* the OS.
- A launcher (⌘K) indexed over apps, projects, commands.
- A theme engine — try `theme set retro` in the terminal.

## the rules I built it under

- No generic dashboard. No reused "card" for major apps.
- Mobile is the same OS, adapted — not a separate site.
- Every app has its own personality.
